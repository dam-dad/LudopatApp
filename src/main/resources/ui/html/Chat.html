Chat multijugador
El chat solamente está disponible para el multijugador online.
Para abrir el chat presione la tecla 'C' o presione el botón "Chat"
situado en la parte inferior izquierda de su pantalla.
Este chat le permite mandar mensajes normales, estilizados, emotes
y mensajes privados que pude usar de la siguiente forma:
- Mensajes normales: Introduzca el mensaje que desee y presione
el botón de enviar (flecha).

- Mensajes estilizados: Todos los estilos han de ponerse al principio
del mensaje y se construyen con un asterisco (*) seguido de un caracter
('b' negrita, 'i' cursiva, 'd' texto negro, 'g' texto dorado) y terminado
de un espacio de forma que un estilo posible sería "*b " (negrita) pudiendo
a continuación introducir el mensaje deseado.

- Emote: Puede pulsar el botón emote (sobre el botón enviar) y luego pulsar
el emote que desee (se enviará automáticamente) o introducir el código
"/emote" seguido de un número del 1 al 8 ambos incluidos.

- Mensaje privado: Introduzca '@' seguido del identificador del jugador al que desee
enviar el mensaje, un espacio y luego el mensaje.
